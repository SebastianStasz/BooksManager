# Books Manager

### About

BooksManager is an application that allows you to search for books on Google Books or eBooks and audiobooks in Apple iTunes.

### Features

- Fetching books informations from google and iTunes API
- Search filtering
- Infinite scrolling
- Custom async image
- DarkMode support

### Technologies

- SwiftUI
- Combine
